# AudioNotes Release Notes

## Version 3.2.8 (2026-05-31)

### 🎉 新功能

#### 📱 桌面小组件支持
- **主屏幕小组件**：快速访问最近的待办事项和录音功能
- **小组件配置**：自定义小组件大小、显示选项和刷新间隔
- **小组件模板**：预配置的小组件模板，适应不同使用场景
- **实时更新**：待办事项变化时自动更新小组件状态
- **背景处理**：优化小组件在后台的性能

#### 📅 系统日历同步
- **双向同步**：待办事项与系统日历事件双向同步
- **冲突解决**：智能处理同步冲突和重复事件
- **权限管理**：完整的日历访问权限请求流程
- **事件映射**：将待办事项映射为日历事件
- **同步状态**：实时显示同步状态和错误信息

#### ✍️ 可选文本输入
- **手动输入模式**：支持语音和文本两种输入方式
- **富文本格式**：支持粗体、斜体、列表等格式
- **快速添加**：简单的文本输入用于快速创建待办事项
- **高级选项**：创建时设置优先级、截止日期和类别
- **输入对话框**：美观的文本输入界面

### 🚀 核心功能增强

#### 🎙️ 语音识别与转录
- **离线语音识别**：完全离线工作，无需互联网连接
- **实时转录**：说话时流式传输部分结果，提供即时反馈
- **语音活动检测**：自动检测停顿分割句子，提高组织性
- **智能待办创建**：将口语想法转换为结构化待办事项

#### 📋 待办事项管理
- **类别分组**：待办事项按类别组织，支持可折叠组
- **优先级管理**：低、中、高优先级分配
- **截止日期**：设置任务截止日期和提醒
- **重复任务**：支持每日和每周重复任务
- **标签系统**：每个待办事项支持多个标签

#### 🎨 用户界面
- **拖放界面**：直观地重新排序类别和待办事项
- **可折叠组**：展开/折叠类别部分，实现专注查看
- **批量操作**：多选和批量操作，提高管理效率
- **视觉反馈**：任务状态、优先级和截止日期的清晰指示器
- **响应式设计**：针对各种屏幕尺寸和方向进行优化

#### 🎛️ 自定义选项
- **模型管理**：下载、切换和管理多个 ASR 模型
- **主题颜色**：调整主题颜色、字体大小和 UI 元素
- **排序选项**：按手动、创建日期、截止日期或优先级排序
- **个性化设置**：通知偏好、默认优先级和 UI 偏好

### 🔧 技术改进

#### 架构优化
- **Clean Architecture**：遵循关注点分离的架构原则
- **Riverpod 状态管理**：响应式状态管理
- **Repository 模式**：数据操作抽象层
- **Provider 模式**：状态提供者模式

#### 性能提升
- **数据库优化**：SQLite 查询优化，支持大型待办列表
- **内存管理**：音频文件内存管理优化
- **UI 响应性**：批量操作时的 UI 响应性改进
- **小组件性能**：小组件后台处理优化

#### 集成增强
- **通知系统**：增强的通知服务，支持多种通知类型
- **安全存储**：敏感数据的安全存储
- **云同步**：WebDAV 客户端支持云同步
- **日历集成**：与系统日历的深度集成
